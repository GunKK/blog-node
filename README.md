# Installation local

* clone repo
```
git clone https://github.com/GunKK/blog-node.git
```
* create .env
```
# mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=express
DB_USERNAME=root
DB_PASSWORD=

server
PORT=3000
```

* run

```
npm i 
npm run start
```
